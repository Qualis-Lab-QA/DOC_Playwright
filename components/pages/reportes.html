Reportes

El framework genera reportes automáticos utilizando Allure Report, una herramienta que permite visualizar los resultados de ejecución de manera clara y profesional, incluyendo capturas, pasos ejecutados, errores y métricas.

Requisitos

  • Es necesario tener Java instalado en el sistema para poder generar y abrir los reportes.
  • También se recomienda tener instalada alguna extensión para servidor local como Live Server en Visual Studio Code para facilitar la visualización.

¿Dónde se generan los reportes?

Al finalizar una ejecución exitosa de las pruebas, se genera automáticamente un reporte dentro de la carpeta: 

output/allure-reports/

Dentro de esa carpeta se encontrará un archivo index.html que puede abrirse para visualizar el reporte completo.

¿Cómo visualizar el reporte de manera local?

Para ver el reporte en tu navegador local, seguí estos pasos:

  1. Abrí Visual Studio Code.
  2. Instalá la extensión Live Server (si no la tenés).
  3. Hacé clic derecho sobre output/allure-reports/index.html y seleccioná "Open with Live Server".
  4. El reporte se abrirá automáticamente en tu navegador por defecto.

Alternativamente, podés ejecutar manualmente: 

allure open output/allure-reports

Esto abrirá una vista temporal del reporte usando el binario de Allure CLI (también requiere Java).

¿Qué incluye el reporte?

  • Resumen general de los escenarios ejecutados.
  • Detalles por cada test: pasos, capturas, errores, duración.
  • Gráficos de ejecución, tasas de éxito/fallo.
  • Adjuntos (screenshots, logs, videos si se configuran).
