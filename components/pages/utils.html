Utils

La carpeta utils/ centraliza funciones auxiliares que son utilizadas en distintas partes del framework. Su objetivo es reutilizar lógica común, evitar duplicación de código y mantener ordenado el proyecto.

¿Qué tipo de utilidades suelen incluirse?

  • Funciones para formateo de fechas y horas.
  • Conversores o manipuladores de strings.
  • Lectura y manejo de datos desde archivos JSON.
  • Generadores de datos para pruebas.
  • Funciones para capturar evidencias o medir tiempos.

Ejemplos comunes

formatearFecha():

Fecha en formato DD/MM/YYYY 
// utils/fecha.js
  export function formatearFecha(fecha = new Date()) {
    const dia = String(fecha.getDate()).padStart(2, '0');
    const mes = String(fecha.getMonth() + 1).padStart(2, '0');
    const anio = fecha.getFullYear();
    return `${dia}/${mes}/${anio}`;
  }

Uso en los steps

// step_definitions/log.steps.js
  import { formatearFecha } from '../../utils/fecha.js';
  
  Then('loggeamos la fecha actual', async () => {
    const fecha = formatearFecha();
    console.log(`🕒 Fecha actual: ${fecha}`);
  });

Ventajas de usar utils

  • Reutilización de lógica común en múltiples archivos.
  • Más claridad en los steps, que quedan enfocados en el flujo.
  • Menos errores y mayor mantenibilidad.
  • Fácil actualización y extensión de funciones compartidas.
